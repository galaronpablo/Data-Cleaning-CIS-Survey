# Data-Cleaning-CIS-Survey
Advanced data cleaning and preprocessing of the CIS survey (Study 3428). Includes outlier detection (LOF), missing data imputation (MICE), and data tidying in R --- Depuración y preprocesamiento avanzado de la encuesta del CIS (Estudio 3428). Incluye detección de atípicos (LOF), imputación de ausentes (MICE) y limpieza de datos en R.
